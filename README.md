# FoodTrack

A lightweight React dashboard for tracking calories, protein, fibre, hydration, Mounjaro-friendly weight goals, food logging, supplements, and strength reminders.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   npm run dev
   ```

## Notes

- Uses Vite + React + Tailwind CSS.
- Main app entry: `src/App.jsx`.
- Dashboard includes weight trend, nutrition stats, food log, habit tracker, and weekly nutrition chart.
